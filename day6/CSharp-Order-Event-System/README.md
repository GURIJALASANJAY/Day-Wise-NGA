# C# Order Event System

## 📌 Description
This project demonstrates a real-time Order Notification System using:
- Delegates
- Multicast Delegates
- Events
- Publisher-Subscriber Pattern

## ⚙️ Features
- Processes an order
- Notifies multiple services:
  - Email Service
  - SMS Service
  - Logger Service
- Uses event-driven architecture
- Loose coupling between components

## 🧠 Concepts Used
- Delegate
- Event
- Multicast Delegate
- Observer Pattern

## ▶️ Sample Output
Order Placed: 101  
Email sent to customer  
SMS sent to customer  
Order logged successfully  

## 🚀 How to Run
1. Compile and run the program
2. Enter order details
3. Observe notifications
